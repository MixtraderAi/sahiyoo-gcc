# SAHIYOO × GCC 2026 — Design System & Visual Direction

## 1. Project Overview

**Project name:** SAHIYOO × GCC 2026 Strategic Dossier  
**Document type:** Premium executive B2B brochure / strategic dossier  
**Format:** Vertical brochure, A4 portrait  
**Audience:** GCC executive stakeholders, engineering/product design delegation, commercial fleet partners, RV modification and NEV infrastructure stakeholders  
**Tone:** Premium, technical, structured, strategic, government/commercial fleet oriented  
**Important rule:** The approved final text must not be rewritten, shortened, edited, paraphrased, or modified. Design work applies only to layout, typography, hierarchy, imagery, colors, spacing, and visual presentation.

---

## 2. Core Design Direction

The brochure should feel like a high-end **executive GCC mobility dossier**, combining:

- Commercial NEV fleet deployment
- RV modification engineering
- GCC regional expansion
- Charging infrastructure and depot-based energy strategy
- B2B / government / infrastructure-level communication

The visual style must be:

- Clean
- Premium
- Technical
- Structured
- White-space driven
- Corporate and executive
- Engineering-focused
- Suitable for GCC mobility and infrastructure stakeholders

Avoid making the document look like:

- A consumer automotive advertisement
- A startup pitch deck with neon colors
- A travel brochure
- A decorative luxury brochure
- A colorful infographic-heavy report
- A generic PowerPoint template

---

## 3. Typography

### 3.1 Primary Heading Font

**Font:** Space Grotesk

Use for:

- Cover title
- Page titles
- Section numbers
- KPI figures
- Strategic callouts
- Table titles
- Major labels
- Large data points

Recommended weights:

- Space Grotesk Bold — main titles and large KPI figures
- Space Grotesk SemiBold — section headings and callouts
- Space Grotesk Medium — smaller labels and table titles

### 3.2 Primary Body Font

**Font:** Inter

Use for:

- Body copy
- Paragraphs
- Tables
- Captions
- Footers
- Bullets
- Metadata blocks
- Regulatory and market data
- Dense informational sections

Recommended weights:

- Inter Regular — body text
- Inter Medium — emphasis and metadata
- Inter SemiBold — table headers and highlighted terms
- Inter Bold — important partner names or critical terms only

### 3.3 Typography Rules

- Use Space Grotesk for visual hierarchy and strong executive presence.
- Use Inter for maximum readability in dense content.
- Keep the typographic system consistent across all pages.
- Avoid using more than these two font families.
- Do not use decorative, serif, handwritten, futuristic sci-fi, or overly rounded fonts.
- Maintain strong contrast between headings and body text.

## 4. Color Palette

### 4.1 Primary Colors

| Role | Name | HEX | Usage |
|---|---|---:|---|
| Primary Brand Color | Corporate Deep Blue | `#005A91` | Frames, section numbers, dividers, accents, table highlights |
| Executive Heading Color | Executive Navy | `#0A2239` | Main headings, KPI figures, high-priority labels |
| Body Text | Graphite Text | `#263238` | Main paragraphs, bullets, table body |
| Background | White | `#FFFFFF` | Main page background |
| Soft Background | Soft Technical Grey | `#F3F6F8` | Cards, table alternate rows, content blocks |
| Lines / Borders | Line Grey | `#D8E1E7` | Table lines, dividers, light frames |
| Secondary Accent | Muted Steel Blue | `#4E7FA3` | Captions, secondary labels, footer text |
| Optional Accent | Desert Gold | `#C7A15A` | Very limited premium GCC accent |

### 4.2 Color Usage Rules

- The overall look should be dominated by white, deep blue, navy, and light grey.
- Use `#005A91` as the primary visual identity color.
- Use `#0A2239` for strong executive titles and main data points.
- Use `#263238` for readable body text.
- Use `#F3F6F8` for subtle section cards and table row backgrounds.
- Use `#D8E1E7` for thin borders and technical grid lines.
- Use `#C7A15A` only as a micro-accent, never as a dominant color.
- Avoid bright green EV clichés, neon colors, rainbow gradients, red/orange warning-style accents, and overly saturated colors.

## 5. Layout System

### 5.1 Format

- Orientation: Vertical / Portrait
- Recommended format: A4 portrait
- Suggested digital ratio: 3:4 vertical or A4 vertical equivalent
- Safe margins: 8–12% from the page edge
- Use consistent header/footer zones across the document

### 5.2 Grid

Use a clean modular grid:

- 12-column grid for flexible page composition
- Strong left alignment for text-heavy pages
- Large white space around key content
- Consistent vertical rhythm
- Structured content cards for dense data
- Avoid overcrowding pages

### 5.3 Page Composition

Preferred composition style:

- Page number / section number in upper-left or upper-right
- Strong page title in the upper area
- Main content in structured blocks
- Key data in KPI cards
- Tables with disciplined spacing
- Footer repeated consistently on every page

### 5.4 Visual Hierarchy

Recommended hierarchy:

1. Section number / page number
2. Main title
3. Subtitle or strategic framing line
4. KPI / critical data points
5. Main text / tables
6. Captions, notes, footers

## 7. AI Image Generation Guidance

### 7.1 General Rule

When generating images with AI, do not ask the AI to render final text. AI-generated text may be distorted or incorrect. Generate only the visual background, then place approved text manually in design software.

### 7.2 Universal AI Background Prompt

Use this prompt for cover background generation:

> Create a premium vertical A4 executive brochure cover background for a GCC strategic mobility dossier. The design must feel like a high-end B2B engineering and government fleet report, not consumer advertising. Use a clean white corporate background with deep blue geometric framing, sharp technical lines, and subtle soft grey grid texture. Include an abstract outline of the GCC region with connected electric mobility corridors linking UAE, Saudi Arabia, Oman, Qatar, Bahrain, and Kuwait. Add refined semi-realistic silhouettes of an electric commercial van, electric pickup, electric bus, RV camper, and depot charging infrastructure in the lower-right area, integrated as a strategic fleet ecosystem. Keep the left side and upper-left quadrant mostly empty for title and metadata overlay. Use the color palette: corporate deep blue, executive navy, graphite grey, white, soft technical grey, muted steel blue, and only a minimal sand-gold accent. Premium, precise, clean, structured, technical, executive, GCC mobility expansion style. No text, no logos, no letters, no numbers, no brand names.

### 7.3 Negative Prompt

> No readable text, no fake typography, no distorted letters, no logos, no flags, no people, no license plates, no colorful gradients, no neon cyberpunk, no green eco cliché, no cartoon style, no excessive desert scene, no busy city background, no luxury sports car advertising.

### 7.4 AI Generation Parameters

**Recommended:**
- Aspect ratio: A4 vertical or 3:4 vertical
- Style: premium corporate technical brochure cover
- Resolution: high-resolution, print-ready
- Composition: left text safe area, visual emphasis lower-right or right side
- Text: none
- Logos: none
- Brand names: none

## 8. Internal Page Design

### 8.1 General Internal Page Style

Internal pages should use:
- White background
- Strong top-level title
- Blue section numbers
- Light grey content cards
- Structured tables
- KPI cards
- Thin divider lines
- Consistent footer
- Generous white space

### 8.2 Page Title Style

**Recommended:**
- Font: Space Grotesk Bold
- Color: `#0A2239`
- Section number: `#005A91`
- Optional subtitle: Inter Medium, `#4E7FA3`

**Example visual hierarchy:**
> **03** > **Regional RV Market Opportunity** > Supporting line / context if required

### 8.3 KPI Cards

Use KPI cards for major market figures.

**Recommended styling:**
- Background: `#F3F6F8`
- Top accent line: `#005A91`
- KPI number: Space Grotesk Bold, `#005A91` or `#0A2239`
- KPI label: Inter Medium, `#4E7FA3`
- Small explanatory text: Inter Regular, `#263238`

*KPI cards should feel precise and analytical, not decorative.*

### 8.4 Tables

**Table style:**
- Header background: `#0A2239`
- Header text: white
- Header font: Inter SemiBold or Space Grotesk Medium
- Body text: Inter Regular, `#263238`
- Country/segment labels: Inter Bold or Space Grotesk SemiBold, `#005A91`
- Alternating row background: `#F3F6F8`
- Borders: `#D8E1E7`

Avoid heavy grid lines. Keep spacing readable.

### 8.5 Bullets

**Bullet style:**
- Use simple round bullets or minimal dash bullets
- Bullet color: `#005A91`
- Text color: `#263238`
- Font: Inter Regular
- Important phrases may use Inter SemiBold
- Avoid decorative icons for every bullet

## 13. Do / Don’t Summary

### Do
- Use white space generously
- Keep the design precise and structured
- Use blue geometry consistently
- Make data easy to scan
- Use Space Grotesk for hierarchy
- Use Inter for readability
- Keep tables clean and executive
- Maintain a premium B2B tone
- Generate AI images without text
- Overlay all approved text manually

### Don’t
- Do not edit the approved copy
- Do not let AI generate final text inside images
- Do not use fake logos
- Do not overuse icons
- Do not use neon colors
- Do not use consumer auto-ad styling
- Do not add people unnecessarily
- Do not use decorative patterns
- Do not clutter the layout
- Do not introduce new unapproved messaging
